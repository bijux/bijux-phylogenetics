Compatibility and alias distribution for `bijux-phylogenetics`.

Install this package if you want the shorter package name and CLI command
while using the same runtime implementation as `bijux-phylogenetics`.

## Install

```bash
python3.11 -m pip install phylogenetics
phylogenetics --help
```

## Compatibility

- runtime implementation is provided by `bijux-phylogenetics`
- public Python API is re-exported under the `phylogenetics` module
- CLI command `phylogenetics` dispatches through the same command handlers as `bijux-phylogenetics`
