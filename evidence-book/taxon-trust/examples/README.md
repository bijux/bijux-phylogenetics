# Taxon Trust Examples

This directory registers durable examples for the `taxon-trust` study.

Each example should keep a stable scientific identity and expose:

- the exact checked-in inputs
- the owning `bijux-phylogenetics` code blocks
- machine-readable observed outputs
- honest per-goal verdicts
