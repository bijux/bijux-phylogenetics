# PCM2 Signal Diagnostics And Open Boundaries

PCM2 also does something important for credibility: it keeps its open edges
visible.

The governed slices cover:

- phylogenetic signal testing
- residual and diagnostic review surfaces
- ancestral reconstruction comparisons
- explicit handling of the remaining intercept-mode likelihood sweep boundary

That last point matters. A useful evidence-book is not only a place where
agreement is documented. It is also where the repository says plainly what is
still incomplete.
