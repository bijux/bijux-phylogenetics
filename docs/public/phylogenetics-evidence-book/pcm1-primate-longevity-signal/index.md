# PCM1: Primate Longevity Signal

PCM1 is the study family built from the Lund lecture material on plots and
phylogenetic signal. In this repository it has been decomposed into smaller
evidence units that track the path from data preparation to comparative signal
and ancestral outputs.

## What PCM1 Is Good For

- understanding the basic tree-and-trait preparation path
- checking that early comparative steps behave consistently across R and Bijux
- teaching what must be true before signal or ancestral results should be
  trusted
