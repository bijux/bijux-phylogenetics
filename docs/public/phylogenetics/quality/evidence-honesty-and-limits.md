# Evidence Honesty And Limits

The public runtime docs should not imply broader validation than the
evidence-book actually proves.

That means:

- the runtime handbook explains the product surface
- the evidence-book explains study-grounded parity and trust
- unresolved gaps stay visible instead of being rounded away in docs language

The result is less flashy, but more credible.
